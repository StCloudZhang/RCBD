🌾 Randomized Complete Block Design (RCBD)

ANOVA + Tukey HSD

📖 What is RCBD? A design that groups experimental units into blocks (homogeneous groups) to remove variability due to nuisance factors. Each block contains all treatments once, allowing us to compare treatments after accounting for block differences.

📐 Statistical Model: Yij = μ + τi + βj + εij

where τi = treatment effect, βj = block effect, εij ~ N(0, σ²).

📐 Sum of Squares Formulas

Method 1: Correction Factor (CF)

CF = G² / N
where G = grand total, N = t × b
SSTotal = ΣY² − CF
SSTreat = [Σ(Tᵢ²)/b] − CF
SSBlock = [Σ(Bⱼ²)/t] − CF
SSError = SSTotal − SSTreat − SSBlock

Method 2: Direct Deviation (No Correction Factor)

SSTotal = ΣᵢΣⱼ (Yij − Ȳ..)²
SSTreat = b × Σᵢ (Ȳi. − Ȳ..)²
SSBlock = t × Σⱼ (Ȳ.j − Ȳ..)²
SSError = ΣᵢΣⱼ (Yij − Ȳi. − Ȳ.j + Ȳ..)²

✔️ Where Ȳ.. = grand mean, Ȳi. = treatment mean, Ȳ.j = block mean. These formulas compute sums of squares directly from deviations without needing a correction factor.

Statistical Note: Tukey's Family-wise Confidence Level

✓ The confidence level (1−α) for Tukey's HSD intervals is family-wise (simultaneous), NOT per-comparison.

✓ Each individual CI has coverage > 1−α (e.g., >95% for α=0.05).

✓ All CIs together have exactly (1−α)×100% confidence that every interval contains its true difference.

✓ No further adjustment (e.g., Bonferroni) is needed — Tukey's method already controls the family-wise error rate (FWER) at α.

📖 Interpretation: With 95% family-wise confidence, if you repeat the experiment many times, 95% of the time all confidence intervals will simultaneously capture their respective true differences.

📖 5. How to Interpret Results

🎯 ANOVA Table Interpretation

  • p-value for Treatments: If p < 0.05 (or your chosen α), reject H₀ → at least one treatment mean differs significantly.
  • p-value for Blocks: If p < 0.05, blocking was effective at reducing variability. If p > 0.05, blocking may not have been necessary.
  • F-statistic: Larger F values (relative to critical F) indicate stronger evidence against the null hypothesis.

🔬 Tukey's HSD Pairwise Comparisons

  • ✅ Significant: The confidence interval for the difference does NOT contain zero → the two treatment means are statistically different.
  • ○ Not significant: The confidence interval contains zero → insufficient evidence to declare a difference.
  • p-value (Tukey-adjusted): Probability of observing such an extreme difference by chance if means were equal, adjusted for multiple comparisons. Smaller p-value = stronger evidence.
  • The HSD value is the minimum absolute difference required for significance at the chosen α.

📊 Practical Example

If Treatment A mean = 10, Treatment B mean = 15, HSD = 3, difference = 5 → significant (5 > 3).
If 95% CI for (A-B) is (2, 8) → does not contain 0 → significant difference.

💡 Note: p-values from ANOVA are unadjusted; pairwise p-values are Tukey-adjusted for multiple comparisons, maintaining family-wise error rate at α.
